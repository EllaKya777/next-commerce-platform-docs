What's Included

Storefront

  • Responsive catalog with dynamic admin-defined filters
  • Full-text search overriding all other filters
  • Product cards with live variant switching
  • Gallery, carousel, tabbed product info
  • Cookie consent + GTM integration

Cart & Checkout

  • MongoDB-persisted cart — 5-day TTL, one per device
  • Smart cart sync on login
  • 5-step checkout + Buy Now bypass
  • Stripe redirect + Cash on Delivery (per country)
  • PDF invoice generated server-side, sent by email

Admin Panel

  • CRUD for Products, Categories, Posts + WooCommerce sync
  • XLSX / JSON import with differential algorithm
  • Live theme editor + translation editor (no rebuild)
  • Per-country delivery profiles
  • Built-in guide documentation — 22 sections

Auth & Security

  • Passwordless only: Google OAuth + Magic Link
  • Middleware pipeline: i18n → IP filter → CSP nonce
  • Zod validation on all API inputs
  • DOMPurify on all rendered content
  • Cloudflare WAF recommended at perimeter

SEO & i18n

  • generateMetadata() + Open Graph on all pages
  • JSON-LD: Organization, BlogPosting, FAQPage, Review
  • Auto sitemap.xml + 410 Gone on deletion
  • Google Shopping XML feed
  • Locale-prefixed routing, managed from admin

Order Management

  • Dual-store: MongoDB + WooCommerce
  • Completed orders retained 6 months in MongoDB
  • Order visibility by Billing email
  • Customer profile with status tracking + invoices
  • E2E Playwright tests for all critical flows
